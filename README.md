# Superteam Projects

A monorepo for production-grade projects built from the Superteam ecosystem's open ideas.

## Projects

| Project | Description | Stack |
| --- | --- | --- |
| [mAIrket](./apps/mairket) | AI-assisted Solana DeFi market forecasts and risk intelligence | Next.js, TypeScript, Recharts |

## Development

```bash
npm install
npm run dev
```

The development server starts at [http://localhost:3000](http://localhost:3000).

## Quality checks

```bash
npm run lint
npm run typecheck
npm run test
npm run build
```
